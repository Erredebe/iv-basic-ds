# Tokens

Los tokens viven en `src/global/tokens.css` y se publican en `/build/iv-basic-ds.css`.

En el modelo Atomic Design, los tokens son foundations transversales: alimentan atoms, molecules, organisms y templates, pero no son componentes Stencil.

## Prefijo

Todos los tokens usan el prefijo `--iv-`.

```css
:root {
  --iv-color-primary: #2563eb;
  --iv-color-text: #102a43;
  --iv-color-background: #ffffff;
  --iv-radius-md: 0.625rem;
  --iv-space-md: 1rem;
}
```

## Tematizacion

Como los componentes no usan Shadow DOM, una aplicacion puede sobrescribir tokens desde CSS global.

```css
:root {
  --iv-color-primary: #7c3aed;
}
```

Storybook incluye un selector de tema claro/oscuro para revisar componentes contra un set alternativo de tokens.
