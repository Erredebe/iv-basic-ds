# Uso Por CDN

El DS se consume cargando el bundle generado por Stencil desde Netlify.

```html
<script type="module" src="https://iv-basic-ds.netlify.app/build/iv-basic-ds.esm.js"></script>
<script nomodule src="https://iv-basic-ds.netlify.app/build/iv-basic-ds.js"></script>
```

Los estilos globales y tokens se publican como:

```html
<link rel="stylesheet" href="https://iv-basic-ds.netlify.app/build/iv-basic-ds.css" />
```

## Ejemplo

```html
<iv-button variant="primary">Guardar</iv-button>
```

## Rutas Publicadas

- `/`: indice de demos Stencil.
- `/demos/atoms/button.html`: demo HTML de Button.
- `/demos/atoms/icon.html`: demo HTML de Icon.
- `/demos/atoms/input.html`: demo HTML de Input.
- `/demos/molecules/dialog.html`: demo HTML de Dialog.
- `/storybook/`: Storybook.
- `/build/iv-basic-ds.esm.js`: bundle ESM.
