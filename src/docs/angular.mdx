# Angular

El consumo principal previsto es Angular usando Custom Elements desde CDN.

## Cargar El CDN

En `src/index.html` de la aplicacion Angular:

```html
<link rel="stylesheet" href="https://iv-basic-ds.netlify.app/build/iv-basic-ds.css" />
<script type="module" src="https://iv-basic-ds.netlify.app/build/iv-basic-ds.esm.js"></script>
<script nomodule src="https://iv-basic-ds.netlify.app/build/iv-basic-ds.js"></script>
```

## Habilitar Custom Elements

Modulo tradicional:

```ts
import { CUSTOM_ELEMENTS_SCHEMA, NgModule } from '@angular/core';

@NgModule({
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
})
export class AppModule {}
```

Componente standalone:

```ts
import { Component, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

@Component({
  selector: 'app-root',
  standalone: true,
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
  template: '<iv-button variant="primary">Guardar</iv-button>',
})
export class AppComponent {}
```

## Eventos

Los eventos de Stencil se escuchan como eventos DOM normales.

```html
<iv-dialog (ivClose)="onDialogClose($event)"></iv-dialog>
```
