# Uso Por CDN

El DS se consume cargando el bundle agregado generado por Stencil desde Netlify.

```html
<script type="module" src="https://iv-basic-ds.netlify.app/build/iv-basic-ds.esm.js"></script>
<script nomodule src="https://iv-basic-ds.netlify.app/build/iv-basic-ds.js"></script>
```

Los estilos globales y tokens se publican como:

```html
<link rel="stylesheet" href="https://iv-basic-ds.netlify.app/build/iv-basic-ds.css" />
```

## Ejemplo

```html
<iv-button variant="primary">Guardar</iv-button>
```

## CDN Versionado Por Componente

Cada paquete de componente publica tambien su propio build versionado:

```html
<link rel="stylesheet" href="https://iv-basic-ds.netlify.app/components/iv-button/0.0.1/build/iv-button.css" />
<script type="module" src="https://iv-basic-ds.netlify.app/components/iv-button/0.0.1/build/iv-button.esm.js"></script>
<script nomodule src="https://iv-basic-ds.netlify.app/components/iv-button/0.0.1/build/iv-button.js"></script>
```

La version actual de Button tambien esta disponible como:

```html
<link rel="stylesheet" href="https://iv-basic-ds.netlify.app/components/iv-button/0.0.2/build/iv-button.css" />
<script type="module" src="https://iv-basic-ds.netlify.app/components/iv-button/0.0.2/build/iv-button.esm.js"></script>
```

Para comparar versiones del mismo tag se usan paginas aisladas o iframes, porque un documento no puede registrar dos implementaciones distintas de `iv-button`.

## Rutas Publicadas

- `/`: indice de demos Stencil.
- `/demos/atoms/button.html`: demo HTML de Button.
- `/demos/atoms/icon.html`: demo HTML de Icon.
- `/demos/atoms/input.html`: demo HTML de Input.
- `/demos/atoms/textarea.html`: demo HTML de Textarea.
- `/demos/molecules/dialog.html`: demo HTML de Dialog.
- `/storybook/`: Storybook.
- `/build/iv-basic-ds.esm.js`: bundle ESM.
- `/components/iv-button/0.0.1/build/iv-button.esm.js`: bundle ESM versionado por componente.
- `/components/iv-button/0.0.2/build/iv-button.esm.js`: bundle ESM actual de Button.
